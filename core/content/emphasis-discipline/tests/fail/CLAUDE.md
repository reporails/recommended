# Project Instructions

## Overview

This is a Node.js microservice handling payment processing.

IMPORTANT: Always use TypeScript strict mode.

CRITICAL: Follow the team style guide at all times.

IMPORTANT: Use ESLint with our custom configuration.

CRITICAL: Never skip the linting step before committing.

IMPORTANT: All functions must have return type annotations.

CRITICAL: Never use `any` type in production code.

## Testing

IMPORTANT: Write tests for all new features.

CRITICAL: Maintain 90% code coverage at all times.

IMPORTANT: Run the full test suite before pushing.

CRITICAL: Never merge with failing tests.

## Deployment

IMPORTANT: Always use Docker for deployments.

CRITICAL: Run migrations before starting the application.

IMPORTANT: Monitor error rates after every deployment.

CRITICAL: Never deploy without a rollback plan.

## Security
